### 🚀 Overview

**TENRA** is a next-generation **MongoDB ORM/ODM** engineered for scalability and real-world enterprise use.
It brings modern features like **dynamic multi-tenancy**, **GraphQL integration**, **Redis caching**, **RBAC**,
and a built-in **CLI scaffolding tool** — all optimized for Node.js.
